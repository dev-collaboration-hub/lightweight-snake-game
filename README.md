# Lightweight Snake Game 🐍

A simple and lightweight Snake game built from scratch with a focus on clean code, performance, and core game development fundamentals.

## 🎯 Project Goal

The goal of this project is to create a fast, minimal, and well-structured Snake game while understanding the internal systems behind games.

## ✨ Features

* Classic Snake gameplay
* Smooth snake movement
* Keyboard controls
* Food spawning system
* Snake growth system
* Collision detection
* Score tracking
* Game over handling

## 🏗️ Architecture

The project focuses on separating core game components:

* Game Loop
* Input System
* Rendering System
* Game State Management
* Collision System
* Score System

## 🛠️ Tech Stack

* Python

## 📌 Vision

Build a clean and lightweight Snake game that can serve as a foundation for learning game development concepts.
